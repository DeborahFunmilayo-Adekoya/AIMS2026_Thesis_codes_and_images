This repository contains the code, processed datasets, trained models, and generated figures used in the analysis for my AIMS 2025 thesis project.

## Repository Contents

The repository includes:

* **R scripts** for converting the colon dataset into `.h5ad` format.
* **Python scripts** used for the colon dataset analysis.
* **Python scripts** used for the lung dataset analysis.
* A collection of **generated figures and visualizations** from the study.
* **Trained model directories** produced during the analysis.
* **Processed datasets** used throughout the project.

This repository serves as a complete archive of the computational workflow and outputs associated with the thesis study.
