# AIMS2025_Thesis_codes_and_images
A collection of all codes used in the analysis of my AIMS Thesis 2025, including all images generated.
This repository contains 4 files; The R script for converting the colon data file to h5ad, The python script with all codes used to run the colon data analysis, the python script used to run the lung data analysis, and a folder containing all images 
generated from the study.
